# 游戏主包模块说明

本文档说明 `assets/bundle/game_main/script` 目录下的模块结构（不包含 `game/demo` 示例代码）。

## 目录结构

```
assets/bundle/game_main/script/
├── Main.ts                    # 游戏主入口
├── base/                      # 基础功能模块
│   ├── Base.ts                # 基础模块管理器
│   ├── advertising/           # 广告模块
│   ├── button/                # 通用按钮模块
│   ├── network/               # 网络模块
│   ├── prompt/                # 提示弹窗模块
│   └── share/                 # 分享模块
├── game/                      # 游戏业务模块
│   ├── account/               # 账号模块
│   ├── common/                # 公共模块
│   ├── guide/                 # 新手引导模块
│   ├── initialize/            # 初始化模块
│   ├── reddot/                # 红点模块
│   ├── setup/                 # 设置模块
│   └── world/                 # 世界模块
└── libs/                      # 外部库
    ├── sdk.js                 # SDK 封装库（编译产物）
    └── sdk.d.ts               # SDK 类型声明
```

## 模块详情

### 基础模块 (base)

| 模块 | 说明 |
|------|------|
| **Base** | 基础模块管理器，统一管理网络、SDK、按钮、提示、分享、广告等基础子模块。 |
| **Network** | 网络模块，包含 HTTP 请求和 WebSocket 连接管理。 |
| **Advertising** | 广告模块，支持激励广告、插屏广告、横幅广告。 |
| **Button** | 通用按钮模块，处理按钮点击事件与交互逻辑。 |
| **Prompt** | 提示弹窗模块，处理游戏内各类提示窗口的显示与交互。 |
| **Share** | 分享功能模块，支持主动分享、自定义图片分享、截图分享、朋友圈分享、右上角转发等。 |

### 业务模块 (game)

| 模块 | 说明 |
|------|------|
| **Account** | 账号模块，处理用户登录、用户信息获取、登录流程等。 |
| **Initialize** | 游戏进入初始化模块，负责热更新与加载默认资源。 |
| **World** | World 模块，主场景或游戏世界管理。 |
| **Guide** | 新手引导模块，支持组件方式绑定引导、步骤恢复、穿透模式与事件模拟模式。 |
| **RedDot** | 红点模块，支持动态红点树结构、红点显示对象注册、红点数据更新。 |
| **Setup** | 游戏设置模块，负责游戏音乐音效等设置。 |
| **Common** | 公共模块，包含游戏单例模块管理器、配置管理和表格数据等。 |
