## Common Errors

```typescript
// ❌ Error - Writing business logic in ViewUI
openMain() {
    this.ent.addUi(VC_Friend_Main);
    this.ent.M_Friend_Main.loadData();  // Prohibited! This is business logic
}

// ❌ Error - Directly operating Model
removeMain() {
    this.ent.removeUi(VC_Friend_Main);
    this.ent.M_Friend_Main.clear();  // Prohibited! Should be handled in B_Friend_Main
}

// ❌ Error - Using oops.gui.open() / oops.gui.remove()
openMain() {
    oops.gui.open('VC_Friend_Main');  // Prohibited! Should use this.ent.addUi()
}
```
