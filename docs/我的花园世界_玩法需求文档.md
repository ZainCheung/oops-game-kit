## 一、游戏定位与核心体验

### 1.1 游戏概述

《我的花园世界》是一款以**"种花 + 开店 + 换装"**为核心的模拟经营类休闲社交游戏。玩家在虚拟花园中从零开始开垦土地、培育数百种鲜花，将收获的花材精心搭配成插花作品出售盈利，同时自由布置装饰花园庭院，体验从一粒种子到繁花满园的治愈养成过程。

### 1.2 核心体验目标

| 体验维度 | 目标描述 |
|---------|---------|
| 收集感 | 数百种花卉从凡品到仙品的收集与升级，满足图鉴收藏驱动 |
| 经营感 | 从种植→花艺创作→订单售卖→花园扩建的完整经营循环 |
| 创意感 | 主题插花、风格化花园布置、自由搭配，满足个性化表达 |
| 治愈感 | 无压力慢节奏、离线成长、花卉生长细节动画与音效 |
| 社交感 | 好友互助浇水/施肥、公会协作、全服共建活动 |
| 成长感 | 等级提升解锁新花种/新系统，花艺技艺从凡品到仙品 |

## 二、核心操作系统

### 2.1 基础操作流程

```
开垦土地 → 平整土壤 → 播种 → 浇水 → 施肥 → 除虫 → 收获 → 出售/制作花艺
```

### 2.2 操作交互需求

- **开垦/种植**：点击空地→选择种子→确认播种。
- **浇水**：点击缺水花田→使用水壶浇水；支持拖动批量浇水（按住图标拖动即可全部浇水/收获）。
- **施肥**：选择肥料→拖放到目标花田。
- **除虫**：点击出现害虫的花田清除。
- **收获**：点击成熟花田→一键收获；支持拖动批量收获。
- **布置/装饰**：拖放装饰品到花园中，支持摆放预览与旋转。

### 2.3 离线机制

- 离线时花种仍按自然节奏生长，回归后可一键收取成熟花朵。
- 长周期花种（如夜光草 8 小时）适合睡前播种，利用离线时间生长。

## 三、核心玩法系统

### 3.1 鲜花种植系统

#### 3.1.1 种植流程

| 步骤 | 说明 |
|------|------|
| 开垦土地 | 初始 2 块土地，用阳光/金币逐步解锁更多（21 级后单块地成本达 10 万金币） |
| 平整土壤 | 平整避免高低不平影响浇水；翻地除草、施入有机肥提升土壤肥力 |
| 播种 | 选择种子播种；粒径大的种子覆土 3-4 厘米，粒径小的覆土 1-2 厘米 |
| 浇水 | 保持土壤湿润，避免积水；雨水天气无需浇水 |
| 施肥 | 基础肥料用于普通花卉，高级肥料用于稀有花卉，速效肥缩短生长周期 |
| 除虫 | 及时清除害虫，避免作物枯萎；营养液提升作物抵抗力 |
| 收获 | 成熟后收获，获取金币与种子碎片 |

#### 3.1.2 花卉品质等级

| 品质 | 名称 | 收益倍率 | 获取难度 |
|------|------|----------|----------|
| 凡品 | Common | ×1（基准） | 新手赠送 |
| 普品 | Normal | ×1.5 | 金币购买/任务 |
| 珍品 | Rare | ×3 | 活动/花坊币兑换 |
| 华品 | Epic | ×4 | 高阶培育/活动 |
| 仙品 | Legendary | ×5 | 变异/限定活动 |

#### 3.1.3 花卉生长周期分类

| 类型 | 成熟时间 | 代表花种 | 种植策略 |
|------|----------|----------|----------|
| 短周期 | 10-20 分钟 | 向日葵（10 min）、小雏菊（20 min） | 日间在线频繁收获 |
| 中周期 | 1-2 小时 | 玫瑰（2 h）、紫绣球 | 碎片时间种植 |
| 长周期 | 6-8 小时 | 郁金香（6 h）、夜光草（8 h） | 睡前/离线种植 |
| 超长周期 | 12 小时+ | 牡丹、昙花 | 高级玩家策略性种植 |

#### 3.1.4 环境影响

- **季节匹配**：春季种樱花、夏季种向日葵，生长速度提升 15%。
- **天气影响**：雨天无需浇水，提升存活率；冬季低温使成熟速度延缓 20%，需放置暖灯摆件维持效率。
- **光照需求**：太阳花需充足光照，部分花卉有特定光照/湿度需求。
- **连续种植加成**：同一地块连续种植同一种花，可提升该花种稀有度，有概率产生变异品种。
- **夜间加速**：凌晨 0 点，作物生长速度提升 50%。

#### 3.1.5 工具系统

| 工具 | 功能 | 升级建议 |
|------|------|----------|
| 水壶 | 浇水 | 优先解锁多功能水壶，减少 50% 浇水次数 |
| 镰刀 | 除草/收获 | 按主线进度升级 |
| 杀虫剂 | 除虫 | 适量储备 |
| 肥料 | 加速生长/提升品质 | 基础肥料用于普通花；速效肥料用于活动限定花 |
| 营养液 | 提升稀有度/抵抗力 | 用于稀有花卉变异概率提升 |
| 阳光增幅器 | 增加 20% 产量 | 用于高价值流种植 |

### 3.2 花艺创作系统

#### 3.2.1 系统概述

将收获的鲜花材料精心搭配创作成插花作品，是连接"种植"与"盈利"的核心环节。

#### 3.2.2 创作流程

1. 选择花瓶/花器。
2. 选择主题（如"春日野餐"、"冬日暖居"、"下午茶"、"生日宴"）。
3. 搭配花材：主花（如玫瑰、昙花）置于视觉中心，辅花（雏菊、金盏菊）环绕。
4. 调整花材角度与高度（角度控制在 15°-30° 更佳）。
5. 完成花艺作品→上架出售或交付订单。

#### 3.2.3 花艺品质影响因素

- 花卉品级（凡品→仙品）。
- 花瓶/花器等级。
- 搭配协调度（色彩、主题匹配）。
- 特殊道具加成（如凝香露可使花朵带动态光晕）。
- 共生组合特效加成。

#### 3.2.4 花瓶系统

| 花瓶 | 花坊币成本 | 说明 |
|------|-----------|------|
| 忽地笑 | 1680 | 前期优先兑换，解锁紫色马蹄莲花艺，非 VIP 售价最高 |
| 小熊花篮 | 自选 | VIP 后重点培养 |
| 曼陀罗华 | 4880 | 全力打造中间曼陀罗花瓶作品，单份收益可达普通 3 倍 |
| 高难度花瓶 | 需要 9 种花材 | 非氪金玩家建议暂不领取 |

### 3.3 花坊系统

#### 3.3.1 系统概述

花坊是花卉交易与花材兑换的核心场所，通过花坊币进行花卉和材料兑换。

#### 3.3.2 花坊币获取

- 完成顾客订单获取花坊币。
- 花坊悬赏任务奖励。
- 活动奖励与兑换码。

#### 3.3.3 兑换优先级策略

1. 先只换**主线任务要求的花卉**，从需要最少材料的开始换。
2. 完成主线要求后，重点培养花坊悬赏给的三种花卉：**红鹤芋、粉乒乓菊、迎春花**，是前期做花艺最赚的。
3. 后续从 4880 花坊币的珍品花卉（深紫底色）开始换。
4. 氪金玩家优先换氪金花瓶所需的花坊花卉。

### 3.5 花园布置系统

#### 3.5.1 区域划分

- **种植区**：按花种类型分区（草本区、木本区、稀有区），参考"一米菜园"方式划分小块。
- **装饰区**：摆放摆件、路径、围栏。
- **休息区**：长椅、遮阳伞等提升氛围感。

#### 3.5.2 装饰风格

| 风格 | 搭配建议 |
|------|----------|
| 森系 | 木质摆件、自然色调 |
| ins 风 | 简约摆件、浅色系 |
| 田园风 | 田园摆件、暖色调 |
| 中式 | 中式屏风、庭院布局 |
| 欧式 | 欧式花钟、对称布局 |

#### 3.5.3 装饰获取方式

- 日常任务与活动奖励。
- 友情值兑换。
- 限定活动解锁（如春日花架、彩虹围栏）。
- 全服花廊活动奖励（永久花境边框）。

#### 3.5.4 布局技巧

- 高株花卉（牡丹、百合）种后排，矮株（太阳花、雏菊）种前排。
- 路径围绕种植区铺设，方便浇水收获。
- 装饰与花卉风格匹配。
- 支持摆放预览，避免遮挡关键区域。

通过 3 种指定花卉组合种植，触发专属共生特效，实现 1+1>2 的收益提升。不同共生体系适配不同运营需求。

| 特效 | "暖阳普照"：三种花卉共享光照需求，成熟时间缩短 40% |

| 项目 | 内容 |

| 项目 | 内容 |

- 旧花种可通过"**共生转化仪**"免费转换为对应共生花种。
