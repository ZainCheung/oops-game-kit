## 一、游戏定位与核心体验

### 1.1 游戏概述

Blooming Town（可译为"繁花小镇"或"花开小镇"）是一款以**"花园栽培 + 花艺设计 + 精品店经营"**为核心的模拟经营类休闲手游，面向海外市场发行。玩家在游戏中从一片宁静的角落开始，将荒芜之地打造为繁花似锦的梦想花园——培育数百种花卉、创作精美插花作品、经营一家时尚花艺精品店，并与邻里好友分享园艺之美。

### 1.2 核心 Slogan（官方）

> "Welcome to Blooming Town, your stylish escape into a world of vibrant color and creative joy."

### 1.3 核心体验目标

| 体验维度 | 目标描述 |
|---------|---------|
| 收集感 | 数百种花卉从种植到收藏的图鉴驱动，持续解锁新品种 |
| 创作感 | 自由搭配花卉与花器，创作个性化花艺作品 |
| 经营感 | 从花园种植→花艺制作→订单交付→店铺扩张的完整经营循环 |
| 治愈感 | 优雅的花园场景、轻松的氛围、无压力的慢节奏体验 |
| 社交感 | 拜访邻居花园、互助互赠、共建温馨花艺社区 |
| 成长感 | 花园不断扩建，解锁稀有花卉品种（如 Lunar Asao clematis） |

## 二、核心玩法系统

### 2.1 花园栽培系统（Nurture Your Dream Garden）

#### 2.1.1 系统概述

花园栽培是游戏最核心的玩法。玩家通过种植、照料、收获数百种独特花卉，将初始空地逐步培育成五彩缤纷的梦幻花园。

#### 2.1.2 已知花卉品种（官方提及）

| 花卉 | 英文名 | 品种特点 |
|------|--------|----------|
| 牡丹 | Peonies | 浪漫优雅，代表性花卉 |
| 大丽花 | Dahlias | 大胆奔放，色彩浓烈 |
| 绣球花 | Hydrangeas | 优雅丰满，花团锦簇 |
| 铁线莲 | Clematis | 攀藤优雅，姿态多样 |
| 月光铁线莲 | Lunar Asao Clematis | 稀有限定品种，后期解锁 |

#### 2.1.3 种植流程（推断）

```
开垦土地 → 选择种子 → 播种 → 浇水/施肥/照料 → 等待生长 → 成熟收获
```

- **土地系统**：初始少量地块，随游戏进度逐步扩建。
- **花卉生长**：不同花卉有不同的生长周期与照料需求。
- **收获**：花卉成熟后收获，获得花材用于花艺创作或订单交付。
- **品种解锁**：随等级/进度解锁更多花卉品种；稀有品种通过特殊条件获取。

### 2.2 花艺设计系统（Design with Effortless Elegance）

#### 2.2.1 系统概述

收获鲜花后进入工作室，将花材与花器搭配，创作精美的插花作品。支持从柔和轻盈到大胆时尚的多种风格。

#### 2.2.2 创作流程

1. 进入工作室（Studio）。
2. 选择花器/花瓶（Vases）。
3. 挑选花材（从已收获的花卉中选择）。
4. 自由搭配与组合。
5. 完成花艺作品→作为中心装饰品展示或出售。

#### 2.2.3 花艺风格方向

- **柔和的（Soft & Airy）**：浅色调、轻盈感、小清新风格。
- **大胆的（Bold & Chic）**：浓烈色彩、视觉冲击、时尚风格。
- **优雅的（Elegant）**：经典搭配、精致高级。

### 2.3 精品店经营系统（Build a Fashionable Boutique）

#### 2.3.1 系统概述

将园艺爱好转化为一门生意——经营一家花艺精品店，为当地居民提供定制花卉服务，将店铺打造为小镇的核心场所。

#### 2.3.2 核心功能

| 功能 | 说明 |
|------|------|
| 定制订单 | 为当地居民完成个性化花卉订单 |
| 特殊活动 | 为特殊场合提供花艺杰作 |
| 店铺成长 | 随业务增长扩建花园、解锁独家花卉品种 |
| 小镇地位 | 店铺逐步成为 Blooming Town 的标志性场所 |

#### 2.3.3 订单系统（推断）

- **居民订单（Custom Orders）**：面向当地居民的日常订单，交付指定花卉或花艺作品。
- **特殊活动订单（Special Events）**：节日、婚礼、庆典等特殊场合的大额订单。
- **订单奖励**：金币、经验、稀有种子/道具。
- **扩张条件**：累计完成订单→提升店铺等级→解锁新区域与新品种。

### 2.4 社区社交系统（Connect in a Warm Community）

#### 2.4.1 系统概述

在温馨的社区中与好友分享花园之美。游戏鼓励玩家之间的友好互动。

#### 2.4.2 社交功能

| 功能 | 说明 |
|------|------|
| 拜访邻居 | 参观好友花园，欣赏他们的创作 |
| 互助帮忙 | 给好友提供帮助（浇水、照料等） |
| 花卉互赠 | 从大方的好友那里获得一些花卉 |
| 社区氛围 | 强调温馨、互助、无竞争的社交环境 |

#### 2.4.3 社交设计理念

- 无排名压力：不设竞技排行榜，避免对比焦虑。
- 互助为主：社交围绕互帮互助而非竞争。
- 温馨社区：强调"Blooming Town is better with good company"。

## 三、经济系统需求

### 3.1 货币与资源（推断）

| 货币/资源 | 用途 | 获取方式 |
|-----------|------|----------|
| 金币（Coins） | 购买种子、基础道具、土地扩建 | 订单交付、花卉出售、活动奖励 |
| 钻石/高级货币 | 购买稀有种子、加速道具、限定装饰 | 内购、成就、活动 |
| 花卉/花材 | 花艺创作、订单交付 | 花园栽培收获 |
| 加速道具 | 缩短花卉生长时间 | 活动、内购、广告观看 |
| 体力/能量 | 执行种植、浇水等操作 | 自然回复、好友互助、广告 |

### 3.2 变现模式

| 方式 | 说明 |
|------|------|
| 广告（Ads） | 激励视频广告（加速生长、额外奖励）、插屏/Banner 广告 |
| 内购（IAP） | 虚拟货币包、限定种子礼包、加速道具、装饰包 |
| 免费下载 | 基础玩法完全免费，付费为加速和美学选项 |

### 3.3 免费与付费平衡

- 核心玩法（种植、设计、订单）完全免费可玩。
- 付费点集中在：加速等待时间、获取稀有/限定花卉、高级装饰。
- 好友互助可替代部分付费加速效果。

## 四、任务与活动系统需求

### 4.1 主线任务

- 引导玩家逐步解锁新功能：花园扩建、花艺工作室、精品店、社交功能。
- 节奏型引导，逐步引入新花卉品种。

### 4.2 日常任务

- 每日种植、收获、完成订单、拜访好友。
- 奖励金币、道具、经验。

### 4.3 特殊活动

- **节日活动**：情人节玫瑰主题、圣诞节冬青主题、复活节花篮等。
- **限定花卉活动**：限时解锁稀有品种（如 Lunar Asao clematis）。
- **社区活动**：好友合作任务、社区共建目标。

### 4.4 成就系统

- 收集类成就：培育 N 种花卉、完成 N 个订单。
- 创作类成就：创作 N 件花艺作品、解锁 N 种风格。
- 社交类成就：拜访 N 个好友、获得 N 次帮助。

## 五、视觉与交互系统

### 5.1 视觉风格

- **主色调**：鲜艳、温暖、充满生机的色彩。
- **风格定位**：优雅时尚的花园美学，强调"stylish escape"。
- **花卉呈现**：每种花卉有独特的外观与生长细节动画。
- **场景氛围**：从宁静角落→繁花似锦的动态变化。

### 5.2 交互设计

- **触屏操作**：单指点击/拖拽完成种植、浇水、花艺搭配。
- **花艺创作**：拖拽花材到花器中，支持自由摆放。
- **花园浏览**：缩放、平移查看花园全景。
